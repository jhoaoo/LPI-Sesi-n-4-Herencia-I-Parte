# LPI · Sesión 4: Herencia I

Repositorio académico de **Lenguajes de Programación I / II** enfocado en el estudio de la **herencia en programación orientada a objetos**.

## Descripción

Este proyecto contiene ejercicios y ejemplos orientados a comprender cómo una clase puede heredar atributos y métodos de otra, aplicando principios básicos de reutilización de código y modelado orientado a objetos.

## Temas trabajados

- Programación orientada a objetos.
- Clases base y clases derivadas.
- Reutilización de atributos y métodos.
- Constructores en jerarquías de clases.
- Organización de código en C++.

## Objetivo académico

Comprender la herencia como mecanismo fundamental de la POO y aplicarla en ejercicios prácticos para resolver problemas de programación de forma estructurada.
